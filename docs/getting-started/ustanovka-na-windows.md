# Установка на Windows

Для установки приложения:

1. Перейдите в раздел загрузок приложения:  
   [https://safekey.cc/#download](https://safekey.cc/#download)
2. Выберите вкладку "Windows"
3. В зависимости от того, нужно Вам устанавливать приложение, или Вы будете запускать приложение из папки (из файла) - нужно скачать **устанавливаемую** или **портативную** версию.

### Устанавливаемая версия

Запустите файл `.exe` и дождитесь установки приложения. После установки приложение запустится автоматически.

Приложение появится в списке приложений. На этом установка приложения закончена.

### Портативная версия

После загрузки файла exe запустите его, приложение будет запущено. Приложение придется запускать из загруженного exe-файла.
