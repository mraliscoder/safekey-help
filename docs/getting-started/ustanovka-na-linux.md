# Установка на Linux

Для установки приложения:

1. Перейдите в раздел загрузок приложения:  
   [https://safekey.cc/#download](https://safekey.cc/#download)
2. Выберите вкладку "Linux"

### Debian / Ubuntu

Скачайте файл `.deb`и выполните команду:\
`dpkg -i /path/to/Safekey-VERSION.deb`

<figure><img src="../../assets/image.png" alt=""><figcaption></figcaption></figure>

Затем запускать приложение можно командой `safekey.`

### AppImage

Загрузите исполняемый файл, предоставьте ему права на выполнение и запускайте файл напрямую.

<figure><img src="../../assets/image (1).png" alt=""><figcaption></figcaption></figure>

### Pacman

Загрузите файл pacman и выполните команду для установки:

`pacman -U /path/to/safekey-VERSION.pacman`

### tar.gz

Загрузите архив, распакуйте его и запускайте приложение напрямую.
