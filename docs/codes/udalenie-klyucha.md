# Удаление ключа

!!! warning "Внимание!"
    После удаления ключа **восстановить его невозможно!**   
    Перед удалением убедитесь, что **отключили** функцию двухфакторной аутентификации в сервисе, ключ для которого вы удаляете.

Для удаления ключа:

1. Нажмите на значок карандаша напротив названия нужного ключа
2. В открывшемся окне редактирования сайта нажмите на красную кнопку мусорного бака
3. Подтвердите удаление кнопкой "Удалить"
