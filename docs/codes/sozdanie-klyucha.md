# Создание ключа

!!! info "Информация"
    На текущий момент добавление ключа через сканирование или загрузку QR-кода недоступно. Данная функция в разработке.

Для добавления нового ключа:

1. Перейдите на вкладку "Добавить" в приложении Safekey
2. Введите понятное и удобное для понимания название ключа. Например, "Sculk ID" или "Госуслуги"
3. Скопируйте и вставьте в поле "Секретный ключ" значение ключа, полученное с сайта, где настраивается двухфакторная аутентификация.
4. Нажмите "Сохранить". Вкладка переключится на "Мои сайты", где будет добавлен ключ.
