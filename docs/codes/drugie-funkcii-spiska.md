# Другие функции списка

Дополнительные функции, доступные из списка ключей:

### Копирование временного кода

Для копирования можно нажать на кнопку копирования напротив необходимого кода. Код будет скопирован.

<figure><img src="../../assets/image (8).png" alt=""><figcaption></figcaption></figure>

### Скрытие кодов

Для безопасности сами временные коды по умолчанию скрыты. Для просмотра какого-либо определенного кода нажмите на значок глаза напротив нужного кода. Код будет показан.

Для скрытия кода нажмите на значок глаза еще раз.

<figure><img src="../../assets/image (9).png" alt=""><figcaption></figcaption></figure>

### Поиск

Для удобного поиска в списке кодов нажмите `Ctrl + F` (`Command + F` на MacOS). В открывшемся окне введите название сайта, коды будут отфильтрованы.
