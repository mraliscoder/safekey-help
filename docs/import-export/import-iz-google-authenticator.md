# Импорт из Google Authenticator

Если у Вас есть желание импортировать ключи с приложения Google Authenticator, выполните следующие действия:

1. Откройте приложение Google Authenticator на iOS / Android
2. В меню приложения нажмите **Перенести аккаунты**
3. Нажмите **Экспортировать аккаунты** и следуйте инструкциям
4. Выберите аккаунты, которые Вы хотите экспортировать. На экране отобразится QR-код.
5. Сделайте скриншот QR-кода или сфотографируйте экран с другого устройства. Сохраните файл QR-кода.
6. В приложении Safekey перейдите на вкладку "Импорт"
7. В разделе "Из QR-кода" нажмите "Выбрать файл" и выберите файл с скриншотом.
8. Если чтение QR-кода прошло успешно, коды будут добавлены в приложение.
