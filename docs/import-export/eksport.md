# Экспорт

Если у Вас есть необходимость перенести ключи на другой ПК, или просто сделать резервную копию, Вы можете воспользоваться функцией **экспорта.**

1. Перейдите на вкладку **Экспорт**
2. Введите пароль шифрования, которым будет зашифрован файл. В случае, если Вы не введете пароль, данные зашифрованы не будут и любой, кто получит файл, **сможет просмотреть секретные ключи**. Настоятельно рекомендуем указывать пароль шифрования.
3. Нажмите **Сохранить** и выберите место сохранения файла.

### Информация о шифровании

Для шифрования используется алгоритм **AES-256.**
