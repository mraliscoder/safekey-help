# Импорт из X2F

Если у Вас есть файл экспорта, полученный по инструкции из статьи [Экспорт](eksport.md), Вы можете его импортировать. Ключи, которых еще нет в приложении, будут добавлены.

1. Перейдите на вкладку "Импорт"
2. В разделе "Из файла X2F" нажмите "Выбрать файл", выберите сохраненный файл формата X2F
3. Введите пароль, если при экспорте данные были зашифрованы
4. Нажмите "Импортировать"
