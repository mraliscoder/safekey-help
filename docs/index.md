# Документация Safekey

### Что такое Safekey?
Safekey - это надежное кроссплатформенное приложение для генерации одноразовых паролей (TOTP) непосредственно на вашем компьютере. Разработанное с фокусом на безопасность и удобство использования, наше приложение помогает защитить ваши онлайн-аккаунты без необходимости использования мобильного телефона.

#### Особенности
* Safekey создает временные коды, которые меняются каждые 30 секунд согласно стандартам TOTP (Time-based One-Time Password).
* Все секретные ключи шифруются локально и никогда не покидают ваш компьютер.
* Многоплатформенность: Поддержка Windows, macOS и Linux.
* Интуитивно понятный дизайн делает управление ключами простым.

### Стоимость программы
Программа распространяется бесплатно.
